# Takeaway 技能

> 当用户发送 "takeaway" 指令时，帮他/她将链接内容存进飞书多维表格

## 触发条件

用户消息包含：
- 关键词：`takeaway`
- 同时包含一个链接（http/https 开头）

## 工作流程

### 1. 提取信息
从用户消息中提取：
- 链接地址
- （如果有）用户提供的标题/分类/总结

### 2. 抓取内容
使用 web_fetch 抓取链接内容，提取：
- 标题（Title）
- 正文内容（主要提取 article 或主要段落）

### 3. 生成总结
使用 LLM 分析内容，生成：
- 主题
- 核心观点
- 关键要点

### 4. 智能分类
根据内容判断分类：
- 实操（教程、指南、步骤类）
- 产品洞察（产品分析、功能解读）
- 行业趋势（行业报告、趋势分析）
- 其他

### 5. 存多维表格
创建记录到飞书多维表格：

| 字段 | 值 |
|------|-----|
| 标题 | 抓取的标题 |
| 分类 | 智能判断的分类 |
| 总结 | 主题 + 核心观点 + 关键要点 |
| 链接 | 原始链接 |

### 6. 反馈用户
告诉用户收录完成！
