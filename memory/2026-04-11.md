# Day 26 - 2026-04-11

## 🎵 Music API 问题解决

**问题：** 音乐生成一直超时失败

**原因：** 旧脚本用 `music-2.5` 模型，但该模型配额为 0/0（无权限）；新模型 `music-2.6` 有 100 条/周配额

**解决：**
- 安装了官方 `mmx-cli`（npm install -g mmx-cli）
- 用 `mmx music generate --model music-2.6` 生成音乐，成功！
- 飞书发送流程：MP3 → ffmpeg 转 opus → 上传 file_type:opus → msg_type:audio 发送

**Key 状态：**
- `MINIMAX_MUSIC_API_KEY` 即是音乐 Key（之前误以为需要单独 Key）
- 用 `mmx auth login` 验证 Key 有效，显示各模型配额

**脚本更新：**
- `send_voice.py` 需要改用 `music-2.6` 模型
- 旧脚本路径：`~/.openclaw/scripts/send_voice.py`

## 📦 技能
