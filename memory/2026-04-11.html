Saturday, April 11, 2026

🎵 Music API 问题解决

  • 问题：音乐生成一直超时失败
  • 原因：旧脚本用 music-2.5 模型，但该模型配额为 0/0（无权限）；新模型 music-2.6 有 100 条/周配额
  • 解决：安装了官方 mmx-cli（npm install -g mmx-cli），用 mmx music generate --model music-2.6 生成音乐，成功！
  • Key 状态：MINIMAX_MUSIC_API_KEY 即是音乐 Key（之前误以为需要单独 Key）
  • 脚本更新：send_voice.py 需要改用 music-2.6 模型

📦 技能

  • mmx-cli 已装为全局工具，支持 text/图像/视频/语音/音乐/搜索
🎵 Music API 修复完成，唱歌功能恢复！
